# CD Archive Management System

A Django-based web application for managing archived CDs and DVDs.

## Features

- Login Authentication
- Dashboard
- Add CD
- View CDs
- Search CDs
- Categories
- View Details
- Edit CD
- Delete CD
- Import Excel
- Export Excel
- Open Folder in Windows Explorer

## Technologies

- Python
- Django
- HTML
- CSS
- SQLite
- OpenPyXL


## How to Run

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
